## FAQ Section

### What is a Telegram Mini App?
Mini Apps are web applications that run inside Telegram. They look and feel native but are actually web pages. You don't need to install anythingâ€”just tap to open.

### Do I need to create a new account?
No. WhatsTask uses your Telegram account for authentication. Just open the Mini App and you're logged in.

### Is my data secure?
Yes. We use industry-standard encryption. Your data is stored securely and never shared with third parties. We don't have access to your Telegram messages.

### Can I use WhatsTask on desktop?
Yes! Telegram Desktop supports Mini Apps. WhatsTask works on mobile and desktop.

### What happens if I exceed API limits?
You'll receive a 429 error. Upgrade your plan or wait for the rate limit to reset (1 minute).

### Can I export my data?
Yes. You can export projects, tasks, and time logs as JSON or CSV.

### Do you have a mobile app?
WhatsTask IS a mobile appâ€”it runs inside Telegram. No separate download needed.
